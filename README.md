# Laboratory 

Requirements to run projects: 
 - nodejs  

## Run and setup

To run lab_1 code: 
```
cd lab_1
node main.js
```


To run lab_2 code: 
```
cd lab_2
npm i
node code.js
```

To run lab_3 code: 
```
cd lab_3
npm i
node index.mjs --file=./corpus/sample.txt
```


To run lab_4 code: 
```
cd lab_4
npm i
node traducere.mjs
```


To run lab_5 code: 
```
cd lab_5
npm i
node index.mjs --iter=5
node index.mjs --iter=7
```
